Place ici tes scripts de création de schéma (schema.sql), modules (modules/...), seeds (seeds/...) pour automatiser la restauration de la base de données de ce projet.

Exemple de structure recommandée :
- schema.sql
- modules/galerie.sql
- seeds/demo_data.sql

Pour bénéficier de l'automatisation, crée un script add_missing_columns_for_seeds.sql adapté si tu utilises des seeds personnalisés.
