## 📝 Exemples d'utilisation

### Modifier ton nom (Section Hero)
1. Clique sur **"Section Hero (Accueil)"**
2. Change le champ **name** : `Ton Nouveau Nom`
3. Clique **"Enregistrer"**
4. ✅ Ton nom change immédiatement en haut de la page !

### Ajouter un article de blog
1. Clique sur **"Blog"**
2. Clique **"Ajouter"** (bouton vert)
3. Remplis :
   ```
   title : Mon nouvel article
   excerpt : Découvrez les dernières nouveautés...
   image : https://images.unsplash.com/photo-xxx
   link : https://monsite.com/article
   date : 5 Octobre 2025
   author : Anthony Legros
   ```
4. Clique **"Enregistrer"**
5. ✅ L'article apparaît avec image et aperçu !

### Ajouter un produit
1. Clique sur **"Boutique"**
2. Clique **"Ajouter"**
3. Remplis :
   ```
   name : Nouveau produit
   description : Description du produit
   price : 99€
   image : https://images.unsplash.com/photo-xxx
   ```
4. Clique **"Enregistrer"**
5. ✅ Le produit apparaît !

### Masquer une section
1. Clique sur n'importe quelle section
2. **Décoche** "Section visible"
3. Clique **"Enregistrer"**
4. ✅ La section disparaît de la page !
